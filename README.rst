====
Crib
====

Crib is a simple command line tool made to encrypt (and decrypt (and open (and edit))) tiny notes and files.

Install
-------

``pip install crib``

Usage
-----

- Encrypt : ``crib -e [filename]``
- Decrypt : ``crib -d [filename]``
- View / Edit : ``crib -s [filename]``
- Help : ``crib -h``
